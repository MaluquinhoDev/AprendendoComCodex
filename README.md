# AprendendoComCodex — Novo Projeto

Projeto reiniciado do zero e preparado para execução de Python no **GitHub Codespaces**.

## Opção 2 (Codespaces) — iniciada neste repositório

Este repositório já inclui configuração em `.devcontainer/devcontainer.json` para abrir no Codespaces com Python pronto.

### Como abrir
1. Acesse o repositório no GitHub.
2. Clique em **Code**.
3. Abra a aba **Codespaces**.
4. Clique em **Create codespace on main**.

### Como executar o exemplo Python
No terminal do Codespaces:

```bash
python3 exemplos/python/01_busca_linear.py
```

Saída esperada:

```text
Valor 19 encontrado no índice 3.
```
